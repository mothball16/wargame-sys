# Roblox Wargame Systems
An in-progress complete refactor of a bundle of milsim combat systems from 2022-2024, in accordance with modern scripting practices.<br>
Test place: https://www.roblox.com/games/73509557461870/wargame-systems

- modded & optimized ACS 1.7.5 🟡
- locker system 🔴

## Installation
- Prerequisites: Rojo, Roblox Studio
1. Clone the repository to your machine
2. Use Rojo to sync the scripts to a Roblox session. default.project.json targets all included frameworks.
3. Navigate to each framework's Shared folder and copy the asset bundle rbxms into each Assets folder. 
**!! Currently, I don't have released models for installation because a lot of stuff is not currently complete. Contact me on Discord @ mothball16 if you want the most recent version of the essential models.**
